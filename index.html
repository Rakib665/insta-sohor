What does js event loop do?

The event loop is activated by sending a request to a local or remote event source. The event loop is a continuous operation that keeps track of the callback queue as well as the call stack. The runtime model of javascript is based on an event loop, which executes the code and collects and processes events, and executes queued sub-tasks.

Local storage VS Session storage?

The data in local storage does not expire, but the data in session storage does when the page session expires.
